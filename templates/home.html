Weather Data API

Get temp from one station on a date: http://127.0.0.1:5000/api/v1/station/date

Example: http://127.0.0.1:5001/api/v1/10/1988-10-25

Get all data from one station: http://127.0.0.1:5000/api/v1/station

Example: http://127.0.0.1:5001/api/v1/10

Get all data from one station on a determinated year: http://127.0.0.1:5000/api/v1/yearly/station/year

Example: http://127.0.0.1:5001/api/v1/yearly/10/1988
